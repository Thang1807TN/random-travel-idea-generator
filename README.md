# Random Travel Idea Generator

## Project Description

A website that generates random travel ideas for users who do not know where to go.

## Main Features

- Generate random destination
- Filter by travel type
- Display destination image, description, and estimated budget

## Travel Types

- Cheap travel
- Nature
- City
- Beach
- Adventure

## Team Roles

- Frontend: UI design, pages, styling, result card
- Backend: API, server, database connection
- Database & Integration: schema, sample data, testing, deployment

## Technologies

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL
- Version Control: GitHub

## Planned APIs

- GET /destinations/random
- GET /destinations?type=beach
